## 超链接

### 网页链接

语法一：

```
使用:  [链接文字](链接地址)

[baidu](http://www.baidu.com)
```

[baidu](http://www.baidu.com)

语法二：

```
定义:  [标识]:链接地址
使用:  [百度][标识]

[link_1]:http://www.baidu.com 
[link_2]:http://www.google.com 

> 欢迎来到[百度][link_1]

> 欢迎来到[谷歌][link_2]
```

[link_1]:http://www.baidu.com 
[link_2]:http://www.google.com 

> 欢迎来到[百度][link_1]

> 欢迎来到[谷歌][link_2]

语法三:

```
使用: <连接地址>

> <http://www.baidu.com/>
```

http://www.baidu.com/

### 图片

语法一:

```
使用:  ![图片](图片地址)

![图片](../../imgs/1.png)
```

![图片](../../imgs/1.png)

语法二:

```
定义:  [标识]:图片地址
使用:  ![图片Alt][标记]

[img_1]:../../imgs/1.png
![百度logo][img_1]
```

[img_1]:../../imgs/1.png
![百度logo][img_1]
